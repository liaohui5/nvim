## introduction

neovim config based [LazyVim](https://github.com/LazyVim/LazyVim) for myself.

## quick start

```sh
# backup your neovim config
mv ~/.config/nvim  ~/.config/nvim-backup

# clone my config
git clone https://github.com/liaohui5/nvim.git ~/.config/nvim
```
